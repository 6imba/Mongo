# When you download MongoDB, it downloads some of MongoDB tools with it.
    - MongoDB Compass.
    - MongoDB Shell.

# MongoDB Tools.
    - MongoDB Compass ==> MongoDB GUI.
    - MongoDB Shell   ==> MongoDB CLI.

# MongoDB Compass GUI:
    > Run MongoDB Compass:
        1. Open MongoDB Compass,
        2. Click on Fill in connection fields Individually,
        3. Press Connect,
        4. Here come your MongoDB Compass DashBoard.
